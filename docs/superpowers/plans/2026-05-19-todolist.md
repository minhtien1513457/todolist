# Todo List App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a React todo list web app with add, complete, and delete functionality, persisted to localStorage.

**Architecture:** Custom hook (`useTodos`) owns all state and localStorage sync logic; three pure UI components (`TodoInput`, `TodoList`, `TodoItem`) receive data and callbacks as props; `App` composes them.

**Tech Stack:** Vite, React 18, Vitest, @testing-library/react, @testing-library/jest-dom
